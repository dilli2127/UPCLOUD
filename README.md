# UpCloud Clone (React)

This is a clone of the [UpCloud](https://upcloud.com/) homepage built with React.

## Setup

1.  Open this folder in your terminal.
2.  Install dependencies (if not already installed):
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm start
    ```

## Features

- **React 18** setup (Manual setup without internal Vite/CRA complexities if they fail).
- **Dark Mode Theme** matching UpCloud's branding.
- **Responsive Design** (Mobile Navigation, Grid Layouts).
- **Components**:
  - Navbar (Sticky, blurred background on scroll)
  - Hero (Animated cloud abstract visual)
  - Social Proof (Logos)
  - Features (Grid layout)
  - Product Grid
  - Footer

## Tech Stack

- React
- CSS Modules / Vanilla CSS variables
- Lucide React (Icons)
